========================컨텐츠

) DB 및 회원가입 : Firebase (구글 제공) 추천 (하지만 이무로이 사이트 특성상 회원가입은 안받는게 좋음)
가장 현실적인 대안.
-장점 : 서버 코딩을 거의 몰라도 구글에서 제공하는 SDK를 자바스크립트에 넣기만 하면, 이메일 회원가입, 비밀번호 암호화 저장, 구글로그인 등을 무료범위 내에서 구현할수 있음
-방식 : 깃허브 페이지(프론트엔드) <-> FireBase (백엔드/DB)
-게임 가볍게 즐기고 가려는데 회원가입은 장벽일수도 있으니 지양하자.
-그냥 익명으로 이름적을수 있게 하는게 나을듯


) 점수 순위 기록 : PlayFab (유니티 공식 파트너) 추천
-유니티 게임 점수 저장에 최적화된 서비스
장점 : 유니티용 SDK 가 잘되어있어서  webGL 게임에서 sendScore(100) 같은 코드를 짜면 자동으로 서버에 저장되고 순위표를 만들어줌.
-비용 : 동시 접속자수가 아주 많아지기 전까지는 무료.


) 방문자 카운트 (혹은 플레이 카운트)
-단순히 지금까지 몇명왔다 보여주는 기능
-Hts 혹은 Flag Counter
-생성된 HTML 코드 한줄을 index.html 하단에 붙여넣기만 하면 끝.

) 방명록
- Giscus (깃허브 기반), 깃허브의 discussions 기능을 사이트로 끌어오는것. 매우 안정적이고 무료, 디자인 깔끔, 댓글 대댓글 이모지반응 다됨. 깃허브 아이디 있으면 누구나 바로 쓸수있음
- Disqus (대중적), 전세계의 수많은 블로그와 사이트에서 쓰는 댓글서비스./ 구글, 페북, 트위터 아이디로 로그인가능. 일반인접근가능. 다만 무료버전은 광고 뜨고, 설정화면이 영어라 진입장벽 조금있음
